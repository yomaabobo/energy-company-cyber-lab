# Energy Company Cybersecurity Homelab

This project simulates the cybersecurity environment of an energy company using a MacBook Pro M2-based homelab.

The goal of this project is to practice cybersecurity skills such as:

- network traffic analysis
- service enumeration
- suspicious activity detection
- incident response documentation

## Lab Environment

The homelab will include:

- Ubuntu Server ARM representing a company system
- Kali Linux ARM representing external testing activity
- Wireshark for network traffic analysis
- Nmap for service discovery

## Project Goal

This project is being developed to learn cybersecurity fundamentals and document the process on GitHub.

Future steps will include:

- creating virtual machines
- analyzing network traffic
- running security scans
- investigating suspicious activity
