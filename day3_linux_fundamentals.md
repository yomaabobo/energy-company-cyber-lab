# Day 3 – Linux Fundamentals

Today I practiced basic Linux investigation commands on the Ubuntu Server VM.

Commands learned:

pwd – displays the current working directory

ls – lists files in the directory

whoami – shows the current logged-in user

uname -a – displays system information

ip a – shows network configuration and IP addresses

These commands are commonly used by cybersecurity analysts when investigating systems.
