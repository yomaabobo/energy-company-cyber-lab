# Day 2 – Virtualization Setup

## Objective
Prepare the lab environment by installing virtualization software and preparing the first virtual machine.

## Tools Introduced

UTM – Virtual machine platform used to run operating systems inside macOS.

Ubuntu Server – Linux server operating system that will simulate a corporate server inside the lab.

## Activities Completed

Installed UTM virtualization platform.

Prepared system for creating the first virtual machine.

## Key Concepts Learned

Virtualization allows multiple operating systems to run on a single physical computer.

Virtual machines simulate real computers and allow cybersecurity professionals to safely test systems and analyze network behavior.
